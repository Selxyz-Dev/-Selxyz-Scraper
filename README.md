### @selxyz/scraper
- Selxyz/scraper Adalah Sebuah Module Yang Di Gunakan Untuk Mendapatkan Berbagai Informasi Dari Berbagai Website.

# Install Module
```bash
$ npm install @selxyz/scraper
```

- Penggunaan Module
```js
(async () => {
const { cnn } = require('@selxyz/scraper')
const data = await cnn()
console.log(data)
})()
```
